Hexo官网：https://hexo.io/zh-cn/index.html/
github官网：www.github.com
Gitee官网: https://gitee.com/

2:安装主题。

现在可以美化一下点击打开：Hexo官网：https://hexo.io/zh-cn/index.html/ 往下拉找到图中“搜索主题”按钮，在打开的网页中寻找喜欢的主题。

打开主题网站后要注意，需要访问github开源仓库需要点击下图红圈的标题链接。
